# 微信公众号排版工具

一个本地可运行的微信公众号图文排版工具。

这一版参考秀米编辑器的思路：不只是换主题，而是把常用公众号排版拆成可插入的模块。

## 功能

- 左侧粘贴文章，右侧实时预览
- 支持 Markdown 基础语法
- 8 套排版主题：极简商务、小红书、知识干货、杂志感、科技蓝、暖橙促销、墨绿质感、暗色高级
- 组件库一键插入：封面导语、标题块、重点卡片、金句卡片、提醒框、步骤清单、双栏对比、按钮、分割线
- 可调字体大小、行距、预览宽度
- 一键复制富文本到公众号后台
- 一键复制 HTML 源码
- 一键导出 HTML 文件
- 自动保存本地草稿
- 不登录、不接数据库、不接公众号 API

## 安装运行

```bash
npm install
npm run dev
```

打开终端里显示的本地地址，例如：

```text
http://localhost:5173
```

## 使用方法

1. 左侧输入或粘贴文章。
2. 选择排版主题。
3. 点击组件库里的模块，一键插入到文章末尾。
4. 右侧实时预览公众号效果。
5. 点击「复制到公众号后台」。
6. 打开微信公众号后台，新建图文，直接粘贴。

## 组件写法说明

工具本质上是 Markdown + HTML 组件混排。

普通内容直接写 Markdown：

```md
# 标题

## 小标题

**重点句**

> 引用内容
```

高级排版可以用内置组件，比如：

```html
<section class="lead-card">
  <p>这里写文章导语。</p>
</section>
```

组件会在预览和复制时自动转成适合公众号粘贴的 inline style。

## 为什么第一版不做自动发布？

公众号自动发布需要接入微信公众号接口，涉及 AppID、AppSecret、IP 白名单、素材上传、草稿箱、发布权限等。

现在先解决最高频的问题：排版、预览、复制。
