# 微信公众号排版工具

一个本地可运行的微信公众号图文排版工具。

第一版只做一件事：

> 把文章 / Markdown 排成适合复制到微信公众号后台的富文本。

## 功能

- 左侧粘贴文章，右侧实时预览
- 支持 Markdown 基础语法
- 3 套排版主题：
  - 极简商务风
  - 小红书风
  - 知识干货风
- 一键复制富文本到公众号后台
- 一键导出 HTML 文件
- 不登录、不接数据库、不接公众号 API

## 安装运行

```bash
npm install
npm run dev
```

打开终端里显示的本地地址，例如：

```text
http://localhost:5173
```

## 使用方法

1. 左侧输入或粘贴文章。
2. 选择排版模板。
3. 右侧预览效果。
4. 点击「复制到公众号后台」。
5. 打开微信公众号后台，新建图文，直接粘贴。

## 为什么第一版不做自动发布？

公众号自动发布需要接入微信公众号接口，涉及 AppID、AppSecret、IP 白名单、素材上传、草稿箱、发布权限等。

第一版先解决最高频的问题：排版、预览、复制。
